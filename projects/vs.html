Vulnerability Scanner

  • Tech Stack

    Flask, Python, SQLite3, HTML, CSS, Bootstrap, JavaScript.

  • Github URL Project Link

This web application is designed to assist cybersecurity professionals in identifying and reporting vulnerabilities in their applications. It uses machine learning algorithms to analyze code and identify potential security breaches. The application provides a user-friendly interface and allows users to upload their code, specify the target platform, and view the identified vulnerabilities and other tools like stignography and other tools.
